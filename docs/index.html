Ejecutar 
porthound

DNS y registros

PortHound puede usar tus propios resolvers para descubrimiento de hosts y consultas inversas. El servidor DNS embebido es ligero y publica registros A y AAAA para laboratorios o nombres internos.

  • Resolvers: udp://, tcp://, dot://, doh://.
  • Entorno: PORTHOUND_DNS_RESOLVERS, PORTHOUND_DNS_TIMEOUT_SECONDS, PORTHOUND_DNS_USE_SYSTEM_RESOLVER.
  • Registros: A, AAAA, CNAME, MX, NS, TXT, SRV, PTR y CAA en tu zona DNS real.

El servidor DNS local suele escuchar en 5300 en laboratorios porque 53 a menudo requiere privilegios de administrador.
