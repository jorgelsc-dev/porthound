DNS and records

PortHound can use your own DNS resolvers for host discovery and reverse lookups. The embedded local DNS server is lightweight and publishes A and AAAA records for labs or internal names.

  • Resolvers: udp://, tcp://, dot://, doh://.
  • Environment: set PORTHOUND_DNS_RESOLVERS, PORTHOUND_DNS_TIMEOUT_SECONDS, and PORTHOUND_DNS_USE_SYSTEM_RESOLVER.
  • Records: use A, AAAA, CNAME, MX, NS, TXT, SRV, PTR, and CAA in your real DNS zone.

The example resolver usually listens on 5300 in labs because port 53 often requires administrator privileges.
