# PortHound Documentation

Official site: [https://porthound.jorgelsc.dev](https://porthound.jorgelsc.dev)

PortHound is a Python network scanner built for authorized security work. It uses `wsbuilder` for the HTTP/WebSocket layer and supports `master`, `agent`, and `standalone` modes.
